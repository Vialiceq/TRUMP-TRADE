# Trump-trade Skill

An skill based on trumpstruth.org/feed that aggregates Trumps public TRUTH Social posts (RSS). It provides educational financial market analysis in Evidence, Latest, Time modes, and Trump-style dialog.

## Features

- Financial market educational analysis based on latest Trump posts  
- Historical queries by user-supplied time ranges  
- Trump-style opinion mode responding to asset questions  
- Evidence mode shows selected Trump posts with detailed financial analysis  
- Includes advanced risk warnings and scenario-based investment commentary  

## Usage Examples

```bash
# Query latest evidence
trump-trade: What did Trump post recently?

# Analyze posts over a time range
trump-trade: Analyze Trumps posts from 2026-04-01 to 2026-04-07

# Trumps view on oil price
trump-trade: Trumps opinion on oil price?

# Chart artist/K-line style analysis
trump-trade: Trump genius / K-line / Oil price, analyze latest
```

## Notes

- For education and research only, not investment advice  
- No buy/sell commands provided  
- Quoted post content must be accurate and non-fabricated
